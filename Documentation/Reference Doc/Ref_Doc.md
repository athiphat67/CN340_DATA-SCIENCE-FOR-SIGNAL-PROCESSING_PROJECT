jrsssb_58_1_267.pdf (Lasso Regression)

ซึ่งงานวิจัยได้อธิบายกลไกสนับสนุนไว้ว่า Lasso Regression ใช้เทคนิค L1 Regularization ที่เพิ่มเทอมการลงโทษ (Penalty term) ในรูปของผลรวมค่าสัมบูรณ์ของสัมประสิทธิ์ ($\lambda \sum_{j=1}^p |\beta_j|$) เข้าไปในสมการวิเคราะห์ความถดถอย ข้อจำกัดทางเรขาคณิตของ L1 นี้จะมีขอบเขตเป็นรูปสี่เหลี่ยมข้าวหลามตัด เมื่อฟังก์ชันความคลาดเคลื่อนต่ำสุดมาสัมผัสกับขอบเขตนี้ มันจึงมักจะไปบรรจบที่บริเวณ "มุม" ซึ่งตั้งอยู่บนแกนพิกัดพอดี กลไกทางคณิตศาสตร์นี้เองที่บังคับให้ค่าสัมประสิทธิ์ ($\beta$) ของตัวแปรที่ไม่มีนัยสำคัญถูกบีบให้กลายเป็น "ศูนย์" อย่างสมบูรณ์ มันจึงทำหน้าที่เป็น Feature Selection ที่คัดกรองตัวแปรที่เป็นเพียงสัญญาณรบกวน (Noise) ทิ้งไปโดยอัตโนมัติครับ

Random Forests จัดการกับปัญหานี้ได้ด้วยการประเมินคุณลักษณะของข้อมูล (features) ผ่านการจำแนกข้อมูลแบบลำดับขั้นที่ไม่เป็นเชิงเส้น (non-linear, hierarchical splits) ยิ่งไปกว่านั้น อัลกอริทึมแบบต้นไม้ตัดสินใจ (decision tree) ยังทำหน้าที่เป็นตัวเลือกคุณลักษณะ (feature selector) โดยธรรมชาติ ซึ่งมันจะเพิกเฉยต่อตัวแปร "สัญญาณรบกวน" (noise) ที่ไร้ความเกี่ยวข้อง ซึ่งมักจะทำให้การพยากรณ์แนวโน้มคลาดเคลื่อน ช่วยให้ทิศทางของแนวโน้มที่แท้จริงโดดเด่นชัดเจนยิ่งขึ้นในผลการทำนาย

เปเปอร์นี้ชี้ให้เห็นว่าโมเดลเชิงเส้น (Linear Models) มักจะมี Bias สูงแต่ Variance ต่ำ ในขณะที่ Decision Tree ทั่วไปมี Bias ต่ำแต่ Variance สูงปรี๊ด (Overfitting) งานวิจัยสรุปว่าไลบรารีสมัยใหม่อย่าง XGBoost และ LightGBM สามารถทำผลงานได้เหนือกว่าโมเดลอื่นๆ บนข้อมูลตาราง เพราะมันสามารถรักษาขอบเขตการตัดสินใจ (Decision boundary) ที่ซับซ้อนไว้ได้ (ลด Bias) ในขณะที่ยังรักษาระยะห่างของ Generalization gap (ความต่างระหว่างความแม่นยำตอนเทรนกับตอนทดสอบ) ให้น้อยกว่า 3% ซึ่งเป็นเครื่องพิสูจน์ถึงความสามารถในการลด Variance

Research_on_gold_price_forecasting_based_on_lstm_a.pdf (Deep learning)

มุมมองจากเปเปอร์: งานวิจัยชี้ให้เห็นว่า ข้อมูลทางการเงินอย่างราคาทองคำนั้นมีความซับซ้อน (Complex) และไม่เป็นเส้นตรง (Non-linear) สูงมาก โมเดลเชิงเส้นแบบ Linear Regression จะพยายามลากเส้นตรงเพื่ออธิบายความสัมพันธ์ ซึ่งมัน "เรียบง่ายเกินไป" ที่จะดักจับโครงสร้างราคาที่ผันผวน

กลไกที่สนับสนุน: ในทางกลับกัน LSTM (Long Short-Term Memory) ถูกออกแบบมาเพื่อแก้ปัญหานี้โดยเฉพาะ โครงสร้างภายในของมันมีสิ่งที่เรียกว่า "Gate" (เช่น Forget gate, Input gate) ซึ่งทำหน้าที่เรียนรู้ว่าข้อมูลราคาในอดีตช่วงไหน "สำคัญและควรจดจำไว้" (เพื่อดูเทรนด์ระยะยาว) และข้อมูลช่วงไหนเป็นแค่ "สัญญาณรบกวน (Noise) ที่ควรลืมทิ้งไป" กลไกนี้ทำให้ LSTM สามารถดักจับโครงสร้างราคาที่ซ่อนอยู่ในการแกว่งตัวของตลาดได้อย่างมีประสิทธิภาพ

การเปรียบเทียบ: ค่า MSE ที่ต่ำระดับนี้เป็นเครื่องยืนยันว่า หากเราต้องการทราบว่า "ราคาจริงน่าจะไปหยุดที่ตัวเลขเท่าไหร่" โมเดล Deep Learning อย่าง LSTM จะเกาะติดกับเส้นราคาจริงได้แนบเนียนกว่า Linear Regression ที่มักจะเกิด Error สะสมมหาศาลเมื่อเทรนด์ราคาเกิดการหักหัวเปลี่ยนทิศทางอย่างรุนแรง

mathematics-11-01985.pdf (Hybrid framework)

งานวิจัยระบุว่า CNN (Convolutional Neural Network) จะทำหน้าที่เป็นด่านหน้าในการกรองสัญญาณรบกวน (Noise) และสกัดเอา "ลักษณะเด่นเฉพาะส่วน (Local Features)" เช่น จุดกลับตัวระยะสั้น หรือกรอบการแกว่งตัว ออกมาจากข้อมูลราคาดิบที่ผันผวน จากนั้นข้อมูลที่ผ่านการสกัดให้สะอาดและมีนัยสำคัญแล้วจะถูกส่งต่อไปยัง Bi-LSTM (Bidirectional LSTM) ซึ่งจะรับหน้าที่วิเคราะห์ความสัมพันธ์ของเวลาแบบ "สองทิศทางขนานกัน" (คือการอ่านข้อมูลจากอดีตมาปัจจุบัน และอ่านย้อนจากปัจจุบันกลับไปหาอดีต) การทำงานร่วมกันเช่นนี้ทำให้โมเดลได้เรียนรู้จากฟีเจอร์ที่ถูกเน้นจุดเด่นมาแล้วจาก CNN ประกอบกับความเข้าใจบริบทแนวโน้มตลาดอย่างลึกซึ้งทั้งเดินหน้าและย้อนกลับจาก Bi-LSTM ส่งผลให้อัตราความคลาดเคลื่อนลดลงอย่างมีนัยสำคัญ และทำนายราคาปิดได้แม่นยำกว่าการใช้โมเดลใดโมเดลหนึ่งเพียงอย่างเดียวครับ
